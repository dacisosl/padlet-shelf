# 업데이트 확인 방법

## 1. 로컬에서 확인

### 방법 1: 브라우저 새로고침
1. 개발 서버 실행 중인 브라우저에서
2. **F5** 키 누르기 또는 **Ctrl + R**
3. 변경사항 확인

### 방법 2: 하드 리프레시
1. **Ctrl + Shift + R** (Windows)
2. 또는 **Ctrl + F5**
3. 캐시를 무시하고 새로고침

## 2. Netlify 배포 확인

### 배포 상태 확인
1. Netlify 대시보드 접속: `app.netlify.com`
2. 프로젝트 선택
3. **Deploys** 탭 클릭
4. 최신 배포 확인:
   - ✅ 초록색 체크 = 성공
   - ❌ 빨간색 X = 실패
   - 🟡 노란색 = 진행 중

### 배포 완료 확인
1. 배포가 완료되면 (초록색 체크)
2. 사이트 URL로 접속
3. **Ctrl + Shift + R** (하드 리프레시)
4. 변경사항 확인

## 3. 버전 확인

### 화면에서 확인
- 헤더에 버전 표시됨
- 예: "패들렛 보드 v2.0"

### 버전 업데이트 방법
1. `src/config/version.js` 파일 열기
2. `APP_VERSION` 값 변경
   - 예: `'v2.0'` → `'v2.1'`
3. 저장 후 커밋 & 푸시

## 4. 변경사항 확인 체크리스트

업데이트 후 확인할 것:
- [ ] 헤더에 버전 번호가 표시되는가?
- [ ] 카드 삭제 버튼(X)이 보이는가?
- [ ] 카드 드래그가 빠르게 작동하는가?
- [ ] 카드 클릭 시 수정 모달이 열리는가?
- [ ] 섹션 추가/삭제/이름변경이 작동하는가?

## 5. 문제 해결

### 변경사항이 반영되지 않을 때
1. **하드 리프레시**: Ctrl + Shift + R
2. **브라우저 캐시 삭제**: 
   - F12 → Network 탭 → "Disable cache" 체크
3. **시크릿 모드에서 확인**
4. **Netlify 배포 로그 확인**

### 배포가 실패했을 때
1. Netlify → Deploys → 실패한 배포 클릭
2. Deploy log 확인
3. 에러 메시지 확인
4. 문제 해결 후 재배포
